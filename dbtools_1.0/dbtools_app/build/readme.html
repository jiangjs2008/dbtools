0. 说明
   提供一个简单的客户端程序，利用Jdbc驱动直接访问数据库，而不用再安装各个数据库厂商提供的客户端
   本程序能够执行SQL脚本，查询、更新或删除数据

1. 运行需求
   必须要有Java运行环境，jre也可，同时在系统环境变量中设定JAVA_HOME或JRE_HOME
   jdk版本要在1.6以上

2. 运行方法
   解压下载的app.zip

2.1 windows系统
    直接运行run.bat即可

2.2 linux系统
    要先给脚本run.sh加上可执行权限
      chmod +x run.sh
    然后才能运行
      ./run.sh

2.3 快捷键的使用
    Alt+F4 关闭程序(系统缺省)
    Alt+x  关闭数据库连接
    Alt+e  执行脚本
    Alt+u  更新数据
    Alt+d  删除数据
    F1     打开帮助文档
    Alt+f  打开File菜单栏
    Alt+v  打开最常用数据库菜单栏
    Alt+b  打开数据库驱动信息菜单栏
    Alt+h  打开帮助菜单栏
